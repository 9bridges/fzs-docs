# 灾备任务

### 功能概述

灾备任务是一种自动故障转移配置。在新建或编辑任务时，您可以指定一个灾备任务。当主任务出现异常时，灾备任务将自动接管同步，保障数据不中断。

### 操作步骤

1. 在数据任务配置界面 `其他配置` 选项卡中，配置灾备任务
 
2. 点击 `提交` 按钮，完成数据任务的添加或编辑。

3. 之后，在数据任务详情界面，点击任务操作，点击菜单中的 `创建` 按钮，创建数据任务。

4. 创建完毕后再点击菜单中的 `启动 Cyrmte` 按钮，启动 Cyrmte（科蓝实时同步组件）。
   :::tip
   主任务和灾备任务都需要分别执行步骤 3 和 4。
   :::
