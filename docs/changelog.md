# 更新日志

此页记录了 FZS 自 `v0.3.0` 以来的的版本更新日志。

## v0.3.0

FZS 于 2023 年 11 月 24 日发布了 `v0.3.0` 版本。

### 功能新增

1. **数据节点可共享 Web Server 配置**：数据节点与 Web Server 不再是一对一的关系，一个 Web Server 可以绑定多个数据节点。此功能使得 FZS 在处理多条同步链路时，减少了 Web Server 的部署数量，不仅降低了系统资源的消耗，也减少了系统管理的复杂度。
2. **数据链路添加更换 FZS 软件包的入口**：当 FZS 软件包需要升级时，可事先「停止」同步链路，通过数据链路的配置界面上传并更新的软件包后，点击「启动」链路来恢复同步。

### 功能优化

1. **数据链路详情页新增 Flow Chart 状态显示**：在数据链路详情页新增了 Flow Chart 状态显示，可直观地查看数据链路的运行状态。
2. **数据节点详情页 UI 布局优化**：数据节点详情页的 UI 布局进行了优化，使得界面更加美观，组件之间的布局更加平衡。

### 功能补丁

1. 「系统设置」入口放到左下角用户信息的右侧。
2. 「主题切换」入口放到左上方的「FZS」图标的最右侧。
3. UI 配色使用 `css variables`，可按客户需求灵活配置。
