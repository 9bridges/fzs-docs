# 编辑数据链路

### 功能概述

修改已有数据链路的配置内容。

### 操作步骤

1. 在数据链路详情界面，点击节点配置，点击菜单中的 `编辑` 按钮。

2. 跳转到编辑数据链路界面，您可以修改数据链路的名称、描述信息、源端和目标端节点，及其他配置信息等。

3. 其余部分参见 [添加数据链路](./add.md)

:::tip

- 运行中的链路无法编辑，请先停止链路。详见 [停止数据链路](./stop.md)
- 编辑链路时，源端和目标端的配置项中会额外显示 Agent 软件包配置。您可以在此将默认的 Agent 替换为自己上传的版本。
