# FZS Web 用户手册

**FZS Web** 是 FZS 数据同步平台的可视化管理中心。通过直观的 Web 界面，用户可以轻松完成从系统部署到日常运维的所有操作。

本手册将引导您完成 FZS Web 的安装配置，并熟练掌握各项功能。

## 内容导航

### 1. 部署与连接
确保您的环境满足要求，并成功通过浏览器访问管理控制台。
* [浏览器兼容性要求](#recommended-browsers)

### 2. [操作指南](./manual/index.mdx)
核心功能的详细操作步骤，包括：
* **数据节点管理**：添加与维护源端和目标端数据库连接。
* **数据链路配置**：创建同步任务，配置全量/增量同步策略。
* **运维监控**：查看任务状态，处理告警与故障。

为了获得最佳的使用体验，建议使用以下浏览器的最新版本：

| 浏览器 | 最低版本要求 | 推荐指数 |
| :--- | :--- | :--- |
| **Google Chrome** | 92+ | ⭐⭐⭐⭐⭐ |
| **Microsoft Edge** | 92+ | ⭐⭐⭐⭐⭐ |
| **Mozilla Firefox** | 90+ | ⭐⭐⭐⭐ |
| Safari | 15.4+ | ⭐⭐⭐ |
| Opera | 78+ | ⭐⭐⭐ |

:::tip 注意
使用的 FZS Web 版本应与 FZS Web Server 版本保持一致，以避免潜在的兼容性问题。
:::
