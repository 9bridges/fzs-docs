# 添加数据链路

### 功能概述

**添加链路** 指在系统中新建一个同步任务配置。在此步骤中，您需要指定同步的源端节点、目标端节点，并配置相应的同步策略。

注意：此操作仅保存配置信息，不会立即启动同步任务。

### 操作步骤

1. **进入添加界面**
   <br/>在「数据链路列表」页面，点击右上角的 **[+ 添加链路]** 按钮。

2. **基本信息配置**
   * **链路名称**：输入一个易于识别的名称（如 `Oracle_to_MySQL_Sales`）。
   * **源端节点**：从下拉列表中选择已注册的源数据库节点。
   * **备端节点**：从下拉列表中选择已注册的目标数据库节点。

3. **高级配置**
   <br/>根据所选的数据库类型，系统会自动加载特定的配置项（如并发数、过滤规则、映射策略等）。请根据业务需求进行填写，大部分选项可以使用默认值。

4. **提交保存**
   <br/>点击 **[提交]** 按钮完成添加。成功后，新链路将出现在列表中，状态为 `未初始`。

:::tip 提示
* 请确保您已提前在 [数据节点管理](../data-node/index.md) 中添加了所需的源端和备端节点。
* 必填项通常标有 `*` 号，其他参数可视情况选填。
:::
