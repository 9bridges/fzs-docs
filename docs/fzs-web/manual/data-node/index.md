# 数据节点

在 FZS Web 中，数据节点是一个数据源的抽象，您可以将一个数据库配置为一个数据节点，然后在 FZS Web 中对该数据节点进行管理和操作。

数据节点是 FZS Web 的基本组成单元，我们支持多种类型的数据库，您可以通过数据节点来定义不同类型数据库的连接信息，且可以定义数据链路的源端节点和备端节点。
